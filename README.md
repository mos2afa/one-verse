One Verse is a simple desktop application that allows you to read the Quran one verse at a time in a clean and distraction-free interface.

## 📥 Download & Run (Release)

1. Go to the **[Releases](https://github.com/mos2afa/one-verse/releases/)** section of this repository.
2. Download the latest release zip file.
3. Extract the zip file to any folder on your computer.
4. Open the folder and double-click on `one verse.exe` to run the application.

> 💡 Tip: Keep the extracted folder in a safe location so the settings file (JSON) is preserved.

## ✨ Features

- Display one verse at a time
- Navigate between verses (Next / Previous)
- Change Surah easily
- Arabic and English support
- Automatically saves the last opened Surah and verse
- Resumes from where you stopped after closing the app

## 🛠️ Built With

- C# - Windows forms
- SQLite (for storing Quran data)
- JSON (for saving user settings)

## 💡 How It Works

The Quran text is stored in a SQLite database.  
User preferences (current Surah and verse) are saved in a JSON file.  
When the application starts, it loads the last saved position automatically.

## 🎯 Goal

The goal of this project is to provide a minimal and focused Quran reading experience — one verse at a time.
